# WELS Frontend

Frontend für das WELS-Projekt, aufgebaut mit Vite und React.

## Technologie-Stack

- React
- Vite
- Tailwind CSS
- MUI (Material UI)
- TypeScript / TSX

## Projekt installieren

```bash
npm install
```

## Projekt lokal starten

Falls du `npm` verwendest:

```bash
npx vite
```

Falls du `pnpm` verwendest:

```bash
pnpm exec vite
```

## Projekt bauen

```bash
npm run build
```

## Ordnerstruktur

- `src/` – Quellcode und Komponenten
- `src/app/` – Haupt-App-Logik
- `src/app/components/` – UI-Komponenten
- `src/app/components/ui/` – UI-Bausteine und wiederverwendbare Controls
- `src/styles/` – globale CSS- und Tailwind-Dateien
- `src/imports/` – statische Assets

## Hinweise

- Achte darauf, dass `node_modules/` nicht im Repository liegt.
- Lokale Umgebungsvariablen sollten niemals in Git eingecheckt werden.
